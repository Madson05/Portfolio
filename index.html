Meus Projetos

Quizzmica

O Quizmica é um jogo de aprendizado de química orgânica, feito em HTML, CSS e JavaScript, com perguntas desafiadoras e feedback imediato. Oferece uma maneira divertida e eficaz de aprender química orgânica.

Delivery-API

A Delivery-API é uma API simples feita com Node.js e Express para gerenciamento de pedidos de entrega. Oferece recursos básicos de CRUD, incluindo leitura e edição de dados. É uma solução fácil de usar para gerenciar seus pedidos.

MiniBlog

O MiniBlog é um blog criado com React no frontend e Firebase no backend. Oferece autenticação, banco de dados (firebase) e CRUD completo para os posts. É fácil de usar e ideal para criar e compartilhar seus posts
